Hướng dẫn nhanh
1. Giải nén vào thư mục webserver (ví dụ C:\xampp\htdocs\linkbio) hoặc /var/www/html/linkbio
2. Đảm bảo PHP có quyền ghi file: file data.json và thư mục uploads phải có quyền ghi.
3. Mở trình duyệt:
   - Trang public: http://localhost/linkbio/index.php
   - Admin: http://localhost/linkbio/admin.php
   - Login: username: admin   password: admin
4. Thay đổi admin: mở file config.php và sửa giá trị 'admin_user' và 'admin_pass'.
5. Avatar upload sẽ lưu vào thư mục uploads/.
6. Dữ liệu lưu trong data.json (JSON, có thể edit trực tiếp khi cần).
